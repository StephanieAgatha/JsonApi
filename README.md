Learning all about json api things ^_^
## API Reference

#### Get all Movies

```http
  GET /lists
```

#### Get item

```http
  GET /movieid
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of item to fetch. Use Form Data |


```http
  GET /moviename
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `name`      | `string` | **Required**. Name of item to fetch. Use Form Data |
